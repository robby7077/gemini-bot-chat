# AI Hotel Chatbot

AI-powered hotel assistant chatbot using Gemini API, Express.js, HTML, CSS, and JavaScript.

## Features

- AI chatbot conversation
- Responsive chat interface
- REST API integration
- Error handling
- Frontend and backend connection

## Technologies Used

- Node.js
- Express.js
- Gemini API
- HTML
- CSS
- JavaScript

## Installation

npm install

## Run Project

npm start

Server:
http://localhost:3000

## API Endpoint

POST:
http://localhost:3000/api/chat

Example JSON Body:

{
  "conversation": [
    {
      "role": "user",
      "text": "halo"
    }
  ]
}
